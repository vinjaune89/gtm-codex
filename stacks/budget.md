# Budget Stack

Free and cheap alternatives per category. For founders, solo operators, and anyone who needs GTM infrastructure without the price tag.

## The Stack

| Layer | Tool | Cost | Notes |
|-------|------|------|-------|
| **CRM** | Attio Free | $0 | Free tier with contacts, companies, deals, and lists. API-first. |
| **Enrichment** | Apollo Free | $0 | 10,000 email credits/month on free tier. Basic enrichment. |
| **Discovery** | Exa AI Free + Octave Free | $0 | Semantic web search + ICP scoring. Both have free tiers. |
| **Scraping** | Apify Free | $0 | $5/month platform credit included. 3,000+ pre-built scrapers. |
| **Outreach** | Manual via Gmail | $0 | Send manually. Or Instantly at $47/month when volume demands it. |
| **Orchestration** | n8n Self-Hosted | $0 | Open-source. Self-host on any server. |
| **AI Agent** | Claude Code | ~$20 | Claude Pro subscription. The highest-ROI spend in this stack. |
| **Content** | Substack | $0 | Free. Newsletter + blog + audience. |
| **Total** | | **$20/month** | |

## Free Tier Gotchas

Every free tier has limits. Know them before you hit them:

| Tool | Free tier limit | What happens when you exceed |
|------|----------------|------------------------------|
| Attio | Core CRM features, limited automation | Upgrade to Plus ($29/seat) |
| Apollo | 10,000 email credits, limited exports | Credits reset monthly. Upgrade for bulk. |
| Exa AI | Limited searches/month | Pay-per-search after that |
| Octave | Limited ICP lookups | Upgrade for higher volume |
| Apify | $5/month platform credit | Pay for compute beyond free credit |
| n8n | Unlimited (self-hosted) | You maintain the server |

## Setup Order

1. **Claude Code** — Your highest-leverage tool. Set up first.
2. **Attio** — Free CRM. Get contacts and deals organized. MCP-native.
3. **Apollo** — Free enrichment. Find emails for prospects.
4. **Exa AI** — Discovery tool. Company/prospect research via MCP.
5. **Octave** — ICP scoring. Qualify prospects programmatically via MCP.
6. **Substack** — Start publishing. Even one post/month builds credibility.
7. **n8n** (optional) — Only if you need workflow automation beyond what Claude Code handles.

## Upgrade Path

When revenue justifies it, upgrade in this order:

1. **Enrichment** → Clay ($149) replaces Apollo. Better data, waterfall enrichment.
2. **CRM** → Attio Plus ($29/seat). More automation, advanced pipelines.
3. **Outreach** → Instantly ($47) when email volume demands it. HeyReach ($79) for LinkedIn.
4. **Signals** → Trigify ($149) for LinkedIn signal detection. Common Room ($1K+) for community.

## Who This Is For

- Pre-revenue founders
- Solo operators testing GTM before committing budget
- Side projects and experiments
- Anyone who wants to validate the approach before scaling spend
