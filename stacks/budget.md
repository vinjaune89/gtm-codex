# Budget Stack

Free and cheap alternatives per category. For founders, solo operators, and anyone who needs GTM infrastructure without the price tag.

## The Stack

| Layer | Tool | Cost | Notes |
|-------|------|------|-------|
| **CRM** | HubSpot Free | $0 | Generous free tier. 1M contacts, deal tracking, email integration. |
| **Enrichment** | Apollo Free | $0 | 10,000 email credits/month on free tier. Basic enrichment. |
| **Research** | Exa AI Free | $0 | Free tier available. Semantic web search. |
| **Sequencing** | lemlist Free or manual | $0–39 | Free trial, then $39/month. Or: send manually via Gmail. |
| **Orchestration** | n8n Self-Hosted | $0 | Open-source. Self-host on any server. |
| **AI Agent** | Claude Code | ~$20 | Claude Pro subscription. The highest-ROI spend in this stack. |
| **Content** | Substack | $0 | Free. Newsletter + blog + audience. |
| **Social** | LinkedIn native | $0 | Post, comment, DM. No paid tools needed. |
| **Total** | | **$20–59/month** | |

## Free Tier Gotchas

Every free tier has limits. Know them before you hit them:

| Tool | Free tier limit | What happens when you exceed |
|------|----------------|------------------------------|
| HubSpot | Limited automation, no sequences on free | Upgrade to Starter ($15/month) |
| Apollo | 10,000 email credits, limited exports | Credits reset monthly. Upgrade for bulk. |
| Exa AI | Limited searches/month | Pay-per-search after that |
| n8n | Unlimited (self-hosted) | You maintain the server |

## Setup Order

1. **Claude Code** — Your highest-leverage tool. Set up first.
2. **HubSpot** — Free CRM. Get contacts and deals organized.
3. **Apollo** — Free enrichment. Find emails for prospects.
4. **Exa AI** — Research tool. Company/prospect research via MCP.
5. **Substack** — Start publishing. Even one post/month builds credibility.
6. **n8n** (optional) — Only if you need workflow automation beyond what Claude Code handles.

## Upgrade Path

When revenue justifies it, upgrade in this order:

1. **Enrichment** → Clay ($149) replaces Apollo. Better data, waterfall enrichment.
2. **CRM** → Attio ($29/seat) replaces HubSpot. Better API, better for agent workflows.
3. **Sequencing** → Instantly ($30) or lemlist ($39). When volume demands it.
4. **Signals** → RB2B (free tier) for website visitor identification.

## Who This Is For

- Pre-revenue founders
- Solo operators testing GTM before committing budget
- Side projects and experiments
- Anyone who wants to validate the approach before scaling spend
